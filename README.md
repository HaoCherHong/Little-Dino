# LittleDino
Make your chrome dinosaur crazy!!

第八次黑客松台灣 - Javascript自動控制Chrome小恐龍

DEMO:
https://www.youtube.com/watch?v=T82g-VHqXh4&feature=youtu.be

# Usage
斷開你的網路，嘗試連上一個網頁，召喚Chrome小恐龍！

將 encoded_url.txt 的內容選取拖曳到書籤列
點擊連結，並按空白鍵開始！
